# Superstore Sales Insights Dashboard

An interactive Excel dashboard providing comprehensive sales analytics and insights for a superstore's performance across multiple dimensions including time, geography, product categories, and customer segments.

## üìä Dashboard Overview

This dashboard transforms raw sales data into actionable business insights through interactive visualizations and key performance indicators. Built entirely in Microsoft Excel, it provides stakeholders with a clear view of sales performance trends and patterns.

## üéØ Key Features

### Summary Metrics
- **Total Sales**: $22,615,536.78 across all periods and categories
- Real-time KPI tracking with dynamic calculations

### Interactive Filters
- **Category Filter**: Furniture, Office Supplies, Technology
- **Region Filter**: Central, East, South, West  
- **Sub-Category Filter**: Detailed product breakdowns including Accessories, Appliances, Art, Binders, Bookcases, Chairs, Copiers, Envelopes, Fasteners, Furnishings, Labels, Machines, Paper

#### 1. Sales by Category (Time Series)
- Multi-year trend analysis (2015-2018)
- Color-coded lines for each product category
- Monthly granularity for detailed pattern recognition

#### 2. Top 5 States Performance
- Horizontal bar chart showing highest performing states
- California leading with ~$4.5M in sales
- Geographic performance comparison

#### 3. Sales by Customer Segments
- Consumer vs Corporate vs Home Office breakdown
- Monthly trend analysis by customer type
- Comparative performance metrics

#### 4. Regional Sales Distribution
- Pie chart showing percentage breakdown by region
- West region leading at 32%
- Visual geographic performance overview

## üìã How to Use

### Getting Started
1. Open the Excel file containing the dashboard
2. Ensure all data connections are active
3. Refresh pivot tables if needed (Data ‚Üí Refresh All)

### Interacting with Filters
1. **Category Selection**: Click on category buttons (Furniture, Office Supplies, Technology) to filter all charts
2. **Region Analysis**: Use region filters to focus on specific geographic areas
3. **Sub-Category Drill-Down**: Select specific product sub-categories for detailed analysis
4. **Reset Filters**: Clear all selections to view complete dataset

### Reading the Charts
- **Time Series**: Hover over data points to see exact values and dates
- **Bar Charts**: Values are displayed at the end of each bar
- **Pie Chart**: Percentages and values shown in legend and slices

## üêõ Troubleshooting

### Common Issues
- **Charts not updating**: Refresh pivot tables (Ctrl + Alt + F5)
- **Filters not working**: Check slicer connections to pivot tables
- **Performance issues**: Reduce data range or optimize pivot table calculations
- **Display problems**: Ensure Excel zoom is set to 100%

### Data Validation
- Verify date formats are consistent
- Check for missing or null values in key fields
- Ensure numeric fields are properly formatted
